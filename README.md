# Gestion d’une plateforme universitaire

## Objectifs
Conception d’un modèle conceptuel de données (MCD) pour gérer étudiants, enseignants, cours, examens et notes.
Implémentation de la base de données via phpMyAdmin.


## Compétences développées 
Modélisation de données, conception de base de données relationnelle, suivi et gestion des inscriptions et résultats.
